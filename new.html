Porsche Car Connect

Type of Work: Mobile Development

Number of Hours: 30+ hours

Software used:

Languages: Swift

Github Repository:

I decided to re-create Porsche’s Car Connect iOS application. I used Adobe XD, Photoshop, and Illustrator to design the layout that you see here. I used Xcode to create the app and used the Firebase API to store user account information. I felt inspired to re-create this application as the "Car Connect" application that was on the market at the time looked horrendous. I thought it could look better that it did.

PrivaSeeable

Type of Work: Web Development

Number of Hours: 12 hours

Software used:

Languages: HTML, CSS, and JavaScript

Github Repository:

Prize: 3rd Place Finalist at CitizenHacks

Privaseeable is a hackathon project that was created in September 2019. It's a chrome extension that was built to maintain a user’s privacy in public settings. My team and I used Javascript to create this chrome extension that takes advantage of CSS elements to maintain user privacy by obsuring their screen’s text from potential onlookers. We implemented a “blur” feature that allows the user to blur onscreen text and seletively unblur text by holding shift and clicking on a piece of text. We implemented a “scramble” feature as well which inserts random words in between onscreen text.

Here's what the chrome extension looks like. There are four options here to choose from.

Here's a video of what this chrome extension looks like in action. Enjoy!
